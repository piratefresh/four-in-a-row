How to port this into Figma

  1. Frames. Create 15 iPhone frames (390×844) matching the screens above — labels on each already map to section numbers.
  2. Colors + type. Copy the tokens above into Figma as Color Styles and Text Styles. File → Preferences → Import variables.
  3. Components. Build LetterTile, ActionButton, CoachBubble, PhoneFrame first as components — everything else reuses them.
  4. Prototype. Wire the Splash → Mode Select → Tutorial steps in order, then branch Post-Tutorial into both Quick Match (→ Signup → Lobby) and Offline (→ Lobby direct).
  5. Flow diagram. Use FigJam or the Flow plugin and recreate the bottom diagram. Each node = one frame from this board.
  6. Screenshot this doc. You can screenshot individual phone frames on this page and drop them into Figma as reference underlays while you rebuild.
