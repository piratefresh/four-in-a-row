# EPIC: Realtime Rooms + Presence + Chat + Signed-In Lobby

## Goal
Ship milestone 1 for poker multiplayer foundation: signed-in lobby listing active tables, room join by code, realtime presence, and realtime chat.

## In Scope
- Signed-in lobby route listing active rooms
- Create room / join room with guest nickname
- Presence updates in realtime
- Realtime room chat
- Cleanup of empty inactive rooms

## Out of Scope
- Poker gameplay logic
- Public unauthenticated lobby
- Moderation and admin tools

## Success Criteria
- Signed-in users can view active room list
- Users can create/join rooms and see live participants
- Chat updates appear live without refresh
- Inactive empty rooms are auto-removed
