# Basic Skill

1. Confirm the user goal in one sentence.
2. Inspect local context before changing files.
3. Make the smallest working change first.
4. Validate the result with a quick check.
5. Report what changed and any next action.

## Notes

- Keep responses concise and actionable.
- Prefer deterministic commands over manual repetition.
